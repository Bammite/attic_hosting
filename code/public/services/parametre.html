Gestion des Expéditeurs d'Email

Ajoutez et gérez les comptes email que vous utiliserez pour envoyer des messages depuis la plateforme.

Ajouter un nouvel expéditeur

Important : Pour utiliser un compte Gmail, vous devez générer un mot de passe d'application. Votre mot de passe habituel ne fonctionnera pas.

  1. Activez la validation en deux étapes sur votre compte Google.
  2. Cliquez ici pour générer un mot de passe d'application.
  3. Sélectionnez "Messagerie" comme application et "Ordinateur Windows" (ou autre) comme appareil, puis copiez le mot de passe généré.

Important : Comme pour Gmail, vous devez utiliser un mot de passe d'application pour Yahoo.

  1. Connectez-vous à votre compte Yahoo et allez dans "Informations personnelles".
  2. Cliquez sur "Sécurité du compte" puis sur "Gérer les mots de passe d'application".
  3. Donnez un nom à votre application (ex: MonCRM) et cliquez sur "Générer". Copiez le mot de passe ici.
